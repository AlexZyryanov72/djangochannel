Разработка интернет магазина на django

Закончив обучение, вы сможете разработать интернет-магазин.

На курсе вы разработаете интернет-магазин который будет состоять из следующих модулей:

  1. Каталог товаров - управляемый из административной панели.

  2. Импорт списка товаров из excel.

  3. Корзина.

  4. Таблица заказов  - в административной панели.

  5. Экспорт заказов в excel.

  6. Поиск.

  7. Фильтр товаров.

  8. Профиль пользователя.

  9. Вход через соц. сети.

А так же будет рассмотрено:

  1. Deploy проекта на сервер.

  2. Настройка nginx.

  3. Подключение сторонних расширений django.

  4. Тестирование приложений django.

В процессе обучения вам поможет ментор. 

Занятия ведет практикующий программист Python с профильным образованием и стажем работы.

Программа курса

Занятия

  • На вебинарах обсуждается: проектирование приложение, иструментарий. Написание части функционала.
  • Одно из занятий в неделю выделяется для обсуждения Вашего кода и даются советы по его улучшению.

Самостоятельная работа

  • После каждого урока вас ждет домашнее задание.
  • В выполнении задания вам помогут ментор или однокурсники.
  • Также вы получите видеозаписи всех занятий, презентации и обучающие пособия.

Проекты в портфолио

Адаптивный frontend интернет-магазина с возможностью регистрации и авторизации
Backend интернет-магазина, позволяющий хранить информацию о товарах, движении средств.

Навыки, которые вы получите

  • научитесь проектировать модели данных
  • научитесь проектировать архитектуры шаблонов и писать template tags
  • узнаете, как создать админку и расширить её возможности
  • приобретёте базовые навыки развёртывания веб-приложений
  • освоите отладку и поиск ошибок
